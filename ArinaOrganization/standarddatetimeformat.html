標準日期時間格式(Standard Date & Time Format)

  標準日期時間格式，是一種被定義為全世界通用、可排序、能夠快速讀取的格式，此處詳細定義標準日期時間的使用方式。

  The standard date and time format is a format that is defined as a universal, sortable, and quickly readable format. Here is a detailed definition of how to use the standard date and time.


  透過使用標準日期時間格式，各式系統能夠統一、有效和快速地進行日期時間的讀存、轉換和排序。

  By using the standard date and time format, various systems can read, store, convert and sort dates and times in a unified, efficient and fast manner.

標準日期時間格式(Standard Date & Time Format) - 固定長度(Fixed Length)

※短日期格式預設為有奈紀年。(Short Date uses Arina date as default.)

擴充格式(Extension Format) - 依照系統需要(According to System Needs)

※短日期格式-擴充預設為有奈紀年。(Short Date EX uses Arina date as default.)

格式說明(Format Instruction)

  • ggg: 以英文字母代表曆法與紀年，共3位含空白，由有奈教發布可使用的名稱，例如目前為 AR(表示有奈紀年)、 CE(表示西元紀年)以及ZZZ(未定義曆法紀年)。

  • yyyyy: 5位整數年份，含負數，不足補0，數值範圍為-9999~99999。

  • yyyyyy: 6位數年份，含負數，不足補0，數值範圍為-99999~999999。

  • MM: 2位整數月份，不足補0。

  • dd: 2位整數日數，不足補0。

  • HH: 2位整數小時，不足補0。

  • mm: 2位整數分鐘，不足補0。

  • ss: 2位整數秒數，不足補0。

  • fffffff: 7位整數表示小數秒數，不足往右補0。

  • ggg: The calendar and era are represented by English letters, a total of 3 letters that include whitespaces. The names that can be used are released by the Arina Organization. For example, " AR" (for the Arina era), " CE" (for the Common era), and "ZZZ" (for the undefined calendar era).

  • yyyyy: The year as a five-digit number, from-9999 to 99999. Add zero if not enough digits.

  • yyyyyy: The year as a six-digit number, from-99999 to 999999. Add zero if not enough digits.

  • MM: The month, 2 digits. Add zero if not enough digits.

  • dd: The day of the month, 2 digits. Add zero if not enough digits.

  • HH: The hour, using a 24-hour clock normally, 2 digits. Add zero if not enough digits.

  • mm: The minute, 2 digits. Add zero if not enough digits.

  • ss: The second, 2 digits. Add zero if not enough digits.

  • fffffff: The ten millionths of a second, zeros are added to the right if digits are not enough.
